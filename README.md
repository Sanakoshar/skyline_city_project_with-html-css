# Skyline City Project

A responsive Skyline City project built using HTML and CSS.

## Overview

This project creates a modern city skyline using various CSS techniques such as:

* CSS Flexbox
* CSS Gradients
* CSS Variables
* Media Queries
* Responsive Design
* CSS Borders and Shapes

The skyline automatically changes from a daytime theme to a nighttime theme on smaller screen sizes.

## Technologies Used

* HTML5
* CSS3

## Features

* Responsive city skyline
* Day and night mode effect
* Gradient-based buildings and sky
* Reusable CSS variables
* Flexbox layouts

## Project Files

* `city.html` - Main HTML structure
* `styles.css` - Styling and responsive design
